# Geometry Works

A recreation of Geometry Dash on Numworks calculators.

Four levels available for now :D

Donwnload the lib: `pip install kandinsky install ion-numworks` and execute `gd.py` to play the game!

## PC Emulator Controls

- OK -> Space
- EXE -> Enter
- UP -> Up Arrow
