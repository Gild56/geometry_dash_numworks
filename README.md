# Geometry Works

A recreation of Geometry Dash on Numworks calculators.

Four levels available for now :D

Donwnload the lib: `pip install kandinsky` and execute `gd.py` to play the game!


## PC Emulator Controls

| Numworks  | Keyboard    |
|-----------|-------------|
| OK        | Space       |
| EXE       | Enter       |
| Backspace | Del         |
| Up        | Up Arrow    |
| Left      | Left Arrow  |
| Right     | Right Arrow |
